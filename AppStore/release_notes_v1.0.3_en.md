# v1.0.3 Release Notes (App Store Submission)

```
• You can now record scores without a video. Use the live stopwatch to log goals during a match, then add the video later and export.

• Edit or delete goal timestamps directly from the event list. PK kick results are now also shown in the list.

• Added penalty timer for sin-bins and suspensions. Display countdown timers per team (for handball, rugby, hockey, etc.).

• Added timeout feature. The match timer automatically pauses during timeouts, and timeout count is displayed on the scoreboard.

• Added restart clock feature. Carry over the timer value seamlessly after halftime.

• Timer can now be placed on either the left or right side of the scoreboard.

• New options for extra-time display: show time in "+MM:SS" format and customize timer text color per segment.

• Improved iPad layout.

• Scores of 10 or more now display correctly on the scoreboard.
```
