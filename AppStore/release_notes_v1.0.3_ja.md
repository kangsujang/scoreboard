# v1.0.3 リリースノート（App Store提出用）

```
• 動画なしでもスコアを記録できるようになりました。試合中にライブストップウォッチで得点を記録し、後から動画を追加してエクスポートできます。

• イベントリストで得点のタイムスタンプ編集と削除に対応しました。PKキックの結果も一覧に表示されます。

• ペナルティタイマーを追加しました。チームごとに退場時間のカウントダウンを表示できます（ハンドボール・ラグビー・ホッケー等に対応）。

• タイムアウト機能を追加しました。タイムアウト中はタイマーが自動停止し、使用回数がスコアボードに表示されます。

• タイマー引き継ぎ機能を追加しました。ハーフタイム後もタイマーを途切れず連続できます。

• タイマーを左右どちらにも配置できるようになりました。

• 延長戦で「+01:23」のようにプラス記号付きで表示するオプションとタイマー文字色の変更に対応しました。

• iPadのレイアウトを改善しました。

• 10点以上の3桁スコアが正しく表示されるようになりました。
```
