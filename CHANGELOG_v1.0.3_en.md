# Version 1.0.3 Changelog

## Score Recording Without Video

- Create a match without selecting a video and go straight to the score editor
- Live stopwatch (start / pause / reset) for real-time recording during a match
- Add a video later and export with the scoreboard overlay

## Event List Editing & Deletion

- Edit the timestamp of any recorded goal or PK kick after the fact
- Delete incorrectly recorded events (with a confirmation dialog)
- PK kick results (◯/✗) are now shown in the event list

## 3-Digit Score Support

- Scores of 10 or more now display correctly on the scoreboard (both preview and exported video)

## Per-Segment Timer Display Options

- Show time in "+MM:SS" format with a plus prefix (for extra time)
- Customize timer text color per segment
- Toggle on/off via "Timer Options" in scoreboard settings

## Timer Position Setting

- Place the timer on the left or right side of the scoreboard
- When placed on the right, the period label also moves to the right of the timer

## Penalty Timer

- Display multiple countdown timers per team simultaneously
- Presets (2 min / 5 min) plus custom duration input
- Displayed outside the scoreboard (below) in a horizontal bar
- Toggle on/off in scoreboard settings

## Timeout Feature

- Record timeout start/end for each team
- Main timer automatically pauses during timeouts
- Timeout count shown as yellow dots next to team names
- Toggle on/off in scoreboard settings

## iPad Layout Fix

- Fixed video preview and score controls overlapping on the score editor screen

## Localization

- Added English, Korean, and Simplified Chinese translations for all new UI strings

## UI Cleanup

- Added toggles in scoreboard settings for penalty timer, timer options, and timeout to hide unused controls
