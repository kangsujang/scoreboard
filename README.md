# ScoreFrame

サッカー動画にスコアボードを合成するiOSアプリ。

## 概要

少年サッカー/草サッカーの試合動画に、後からスコアボードをオーバーレイ合成してYouTubeなどにアップロード可能にするアプリです。

## 技術スタック

- Swift / SwiftUI (iOS 17+)
- SwiftData (永続化)
- AVFoundation (動画合成)
- PhotosUI (動画選択)

## セットアップ

1. Xcodeで新規プロジェクトを作成（App → SwiftUI → ScoreFrame）
2. `ScoreFrame/ScoreFrame/` 配下のソースファイルをプロジェクトに追加
3. Info.plistのプライバシー設定を確認
4. iOS 17.0以上をデプロイメントターゲットに設定
5. ビルド & 実行

## 機能

- 試合の作成・管理
- カメラロールからの動画インポート
- 動画再生しながらリアルタイムで得点記録
- スコアボードのカスタマイズ（位置/テーマ/フォント/タイマー）
- スコアボード付き動画のMP4エクスポート
- 共有 / 写真ライブラリへの保存
