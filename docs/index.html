How it works

4ステップで完成

撮影した動画を選んで、スコアを記録するだけ。簡単な操作でプロのような試合動画が完成します。

動画を選択

写真ライブラリまたはファイルから試合動画を選択

スタイル設定

テーマ・チームカラー・位置を自由にカスタマイズ

スコア記録

動画を再生しながらゴールのタイミングを入力

エクスポート

MP4で書き出して共有・保存

リアルタイムで
スコアを記録

動画を再生しながら、ゴールのタイミングでボタンをタップ。前半・後半・延長戦・PK戦にも対応。スコアボードが動画の時間軸に合わせて自動的に更新されます。

自分だけの
スコアボード

4つのテーマから選び、チームカラーを設定。ピンチで大きさを変え、ドラッグで位置を調整。大会名や日程も表示できます。

試合の全記録を
一目で確認

ゴール、キックオフ、ピリオド終了など、すべてのイベントをタイムライン形式で表示。試合の流れが一目でわかります。

iPad Ready

iPadの大画面でも

横画面レイアウトで、動画とスコアコントロールを同時に表示
